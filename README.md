# Road to success:

## 1. Todo App

## Features:

- create a todo
- list the created todos
- delete todos
- save to local storage

# Stack:

- React
- Typescript
- SCSS modules
- Vite
- Prettier
- ESLint

# Live preview

https://road-to-success-todo-app.netlify.app/

## Installation

Clone the repo and run `yarn install`

## Start

After the successful installation of the packages: `yarn dev`
